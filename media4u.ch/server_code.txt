✅ Prompt für neue React-Komponenten auf Senior-Level

Erstelle eine React-Komponente auf Senior-Niveau, die:

– Inhalte aus einer Strapi-API lädt (z. B. /api/[endpoint]?locale=xx)
– die Sprache via i18n.language?.split('-')[0] || 'de' berücksichtigt
– AbortController nutzt, um Fetch sauber abbrechen zu können
– sanitizeHtml verwendet (aus utils/sanitizeHtml.js, mit DOMPurify), um HTML gegen XSS zu schützen
– Ladezustand, Fehler und Erfolg unterscheidet, inkl. semantischem Markup und a11y (role="alert", aria-live, aria-busy)
– die API-Antwort über resolveContent(data) extrahiert, um bei Strukturänderungen resilient zu bleiben
– nur React-Hooks (useEffect, useState, useMemo) nutzt, keine externen Zustandsbibliotheken
– kein direktes innerHTML oder eval-ähnliche Operationen einsetzt
– HTML nur nach vorheriger Sanitization mit dangerouslySetInnerHTML rendert
– CSS scoped mit import '../../css/[komponente].css' einbindet
– auf ein Vite-Projekt abgestimmt ist (import.meta.env.VITE_STRAPI_URL für die API-Basis)

🔁 Struktur der Komponente:

– useEffect() für Fetch + Cleanup
– useState() für content, error, loading
– useMemo() für sanitizedHtml
– Rückgabe: loading-, error- und Erfolgs-Zustände klar getrennt

🔗 Beispiel-Endpunkte:

– /api/slogan?locale=xx
– /api/impressum?locale=xx
– /api/datenschutz?locale=xx
– /api/about?locale=xx

Nutze nur react, react-i18next, DOMPurify und Vite-Umgebungsvariablen.

  index index.html;

  # HTTPS-Zertifikat (Let's Encrypt – managed by Certbot)
  ssl_certificate /etc/letsencrypt/live/media4u.ch/fullchain.pem;
  ssl_certificate_key /etc/letsencrypt/live/media4u.ch/privkey.pem;
  include /etc/letsencrypt/options-ssl-nginx.conf;
  ssl_dhparam /etc/letsencrypt/ssl-dhparams.pem;

  # Security-Header
  add_header Strict-Transport-Security "max-age=63072000; includeSubDomains; preload" always;
  add_header X-Content-Type-Options "nosniff" always;
  add_header X-Frame-Options "DENY" always;
  add_header Referrer-Policy "strict-origin-when-cross-origin" always;
  add_header Permissions-Policy "camera=(), microphone=(), geolocation=()" always;

  # 🟢 STATIONÄRE SEITEN – /projekte/
  location ^~ /projekte/ {
    root /var/www/vhosts/media4u.ch/httpdocs;
    index index.html;
  }

  # 🔒 STATIONÄRE SEITEN – /editor_2025/ mit Passwortschutz
  location ^~ /editor_2025/ {
    root /var/www/vhosts/media4u.ch/httpdocs;
    index index.html;

    auth_basic "Editor Zugriff (geschützt)";
    auth_basic_user_file /etc/nginx/.htpasswd_editor;
  }

  # SPA (React): fallback für alle übrigen Pfade
  location / {
    try_files $uri /index.html;
  }

  # Assets mit Cache-Header
  location ~* \.(?:js|css|png|jpg|jpeg|gif|svg|ico|ttf|woff|woff2|json|webmanifest)$ {
    expires 6M;
    access_log off;

  }
