🖼️ media4u.ch – Frontend & Art Gallery
Dieses Verzeichnis enthält die Kern-App von media4u.ch. Der Fokus liegt hier auf der Bereitstellung kreativer Inhalte für ein internationales Publikum unter Berücksichtigung von Performance und Datenschutz.

🚀 Highlights
Internationale Skalierung: Vollständige Lokalisierung in 24 Sprachen.

Resiliente API-Integration: Robustes Data-Fetching vom Strapi Headless CMS unter konsequenter Nutzung von AbortController zur Vermeidung von Memory Leaks und Race Conditions.

Datenschutz: Cookiefreie Analytics via Self-Hosted Umami und serverseitige GeoIP-Lokalisierung über MaxMind-Datenbanken.

🎨 Digitale Kunstformen
SMS-Textbilder: Detaillierte Kunstwerke aus Zeichen, optimiert für moderne Rendering-Engines.

Textbild 2.0: Multilayer-Typografie mit variablen Schriftgrößen, Farben, Rotationen und komplexen Grid-Layouts.

ASCII-Art-Film: Frame-basierte Text-Animationen (z. B. Szenen aus der Matrix-Trilogie) mit minimalem Payload.

🛠️ Tech Stack
React & Vite.

Semantisches HTML5 & CSS3 (Modulare BEM-Methodik).

Strict XSS-Protection via DOMPurify.
